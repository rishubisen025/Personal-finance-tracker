
# 💰 Personal Finance Tracker (CLI)

A command-line based Personal Finance Tracker built using Python.  
This project helps users manage monthly budgets, track expenses, and analyze spending patterns with smart insights.

## 🚀 Features

- 📋 Monthly budget setup (category-wise)
- 💸 Add and track daily expenses
- 📜 View all expenses with date & description
- 📊 Monthly report (Budget vs Actual)
- 💡 Smart savings tips (20% rule, max spending category)
- 🗑️ Delete last expense option

## 🛠️ Tech Stack

- Python
- JSON (for data storage)

## ▶️ How to Run

```bash
personal finance tracker
